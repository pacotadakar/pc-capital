⚙️ Configurar Sincronización en la Nube

Para que el dashboard se sincronice entre tu Mac y tu iPhone necesitás crear un bin gratuito en JSONBin.io. Solo hacés esto una vez.

  1. Entrá a jsonbin.io y creá una cuenta gratis
  2. Creá un nuevo bin con el contenido {}
  3. Copiá el BIN ID (ej: 6650a3...)
  4. Creá una API KEY desde tu perfil
  5. Pegá ambos datos abajo 👇
